# MedHistory


## Tech Stack
## 🖥️ Frontend
- **Framework:** React.js 
- **UI Library:** CSS

## 🌍 Backend
- **Server:** Django/Node.js (Express.js for API handling)
- **Database:** mongoDB
- **Encryption** AEFS-256
- **Hosting:** Vercel
  
## 🧠 AI & OCR
- **Text Extraction:** Google Vision API / Tesseract.js (OCR)
- **Image Preprocessing:** OpenCV.js (for denoising & deskewing images)....for ocr

## 🛠️ Dev Tools
- **Version Control:** Git & GitHub
